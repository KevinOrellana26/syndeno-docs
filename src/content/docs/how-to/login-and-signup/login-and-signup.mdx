## Sign Up

Para crear una cuenta en la plataforma, sigue los pasos detallados a continuación: 

<Steps>

1. **Introduce una dirección de correo electrónico válida:**
    <a href="/src/content/docs/img/how-to/login-and-signup/correo.png" target="_blank">
        <img src="/src/content/docs/img/how-to/login-and-signup/correo.png" alt="correo"></img>
    </a> 
2. **Revisa el correo con el código de verificación:**
    <a href="/src/content/docs/img/how-to/login-and-signup/cod_verificacion_mail.png" target="_blank">
        <img src="/src/content/docs/img/how-to/login-and-signup/cod_verificacion_mail.png" alt="cod_verificacion_mail"></img>
    </a>
3. **Introduce el código de verificación:**
    escribe el código recibido número por número y haz clic en el botón **verificar correo electrónico**.
    <a href="/src/content/docs/img/how-to/login-and-signup/intr_cod.png" target="_blank">
        <img src="/src/content/docs/img/how-to/login-and-signup/intr_cod.png" alt="intr_cod"></img>
    </a>
4. **¡Registro completado!:**
    el equipo de soporte se pondrá en contacto contigo lo antes posible para proporcionarte la contraseña de acceso a la plataforma.
    <a href="/src/content/docs/img/how-to/login-and-signup/msj.png" target="_blank">
        <img src="/src/content/docs/img/how-to/login-and-signup/msj.png" alt="mensaje"></img>
    </a>
</Steps>
