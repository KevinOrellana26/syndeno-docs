En la plataforma de Syndeno, crear un ticket de soporte es rápido y sencillo. Si necesitas ayuda o quieres reportar un problema, aquí te mostramos cómo hacerlo.

## Instrucciones para crear un ticket

<Steps>

1. Accede al icono de soporte.
    * En el menú superior derecho de la plataforma, localiza y haz clic en el icono de soporte. Esto abrirá la página de soporte de Syndeno, donde podrás crear un nuevo ticket.
    <br/>
    <a href="/src/content/docs/img/soporte/soporte.png" target="_blank">
        <img src="/src/content/docs/img/soporte/soporte.png" alt="soporte"></img>
    </a>
    <br/> 
2. Selecciona el tipo de solicitud.
    * Una vez dentro de la página de soporte, verás varias opciones de solicitud en pantalla:
        * **Request New Environment:** para solicitar un nuevo entorno, como un entorno de staging.
        * **Request New Pipeline:** para pedir la creación de una nueva base de datos, una aplicación u otros servicios de pipeline.
        * **Report a System Problem:** para notificar sobre problemas técnicos que afectan el funcionamiento de la plataforma.
        * **Get Help:** para obtener asistencia general. 
        * **Modify Existing Pipeline:** para hacer cambios en un pipeline, como agregar variables, modificar limites de CPU o Memoria.
    <br/>
    <a href="/src/content/docs/img/soporte/lista-op.png" target="_blank">
        <img src="/src/content/docs/img/soporte/lista-op.png" alt="opciones"></img>
    </a>
    <br/>
    Selecciona la opción que mejor describa tu solicitud.
3. Completa el formulario de solicitud.
    * Tras seleccionar el tipo de solicitud, se abrirá un formulario específico para ese tipo de ticket. **Completa los campos obligatorios**: 
        * **Raise this request on behalf of:** Selecciona a la persona o equipo que representa el ticket.
        * **Summary:** Proporciona un breve resumen de tu solicitud o problema.
    <br/>
    <a href="/src/content/docs/img/soporte/ticket-general.png" target="_blank">
        <img src="/src/content/docs/img/soporte/ticket-general.png" alt="ticket"></img>
    </a>
    <br/>
4. Enviar el ticket.
    * Una vez completado el formulario, haz clic en **Send** para enviar tu solicitud. Se creará el ticket y el equipo de soporte se pondrá en contacto contigo para resolver tu solicitud.
</Steps>
